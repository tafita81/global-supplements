## 📞 SUPORTE E TROUBLESHOOTING

### **Se tiver problemas:**

1. **Erro ao deployar Edge Functions**
   → Ver `EDGE_FUNCTIONS_SETUP.md` seção Troubleshooting

2. **Testes retornando `mock: true`**
   → Verificar secrets no Supabase Dashboard

3. **Deploy GitHub Actions falha**
   → Verificar secret `FTP_PASSWORD`

4. **Email não chega (SendGrid)**
   → Verificar spam, configurar domínio

5. **GSC retorna erro OAuth**
   → Regenerar refresh_token no OAuth Playground
